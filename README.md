# Chat-Bot Backend

Repository for backend of interactive chat with automated answers. There is no acces to AI interaction.

#### Stack/dependecies used in project:
- node.js
- express.js
##### To run project localy, clone repository:
```
git pull 
```
##### Install dependecies
```
npm install 
```
